Homework 4

The url 'https://jsonplaceholder.typicode.com/users' has a list of users in JSON format. Please type the url in your browser to see the data. This assignment asks you to use two APIs to retrieve data fromteh url. The first API is the old XMLHttpRequest and its callback syntax. The second is the new fetch method and its promise syntax. When access your web page, just display the following two results (in italic blue format) - format the results whatever you want. Tips: use Chrome dev tools to debug / test your code interactively.

Callback

Use the Old XMLHttpRequest method and its callback syntax to retrive the data from teh above url and display the list of user emails - the results are sorted by alphabet order.

Promise

Use the new fetch method and its promise snytax to display the list of user's usernames. The result is sorted by the username length - shortest first.
